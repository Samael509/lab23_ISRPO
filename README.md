## Описание игры

### Правила игры

- Два игрока ходят по очереди: X и O
- Каждый игрок ставит свой символ в пустую клетку
- Побеждает тот, кто первым заполнит три клетки подряд (по горизонтали, вертикали или диагонали)
- Если все клетки заполнены и никто не победил - ничья

### Логика победы

- Проверяются все 8 выигрышных комбинаций после каждого хода
- Если три клетки одной комбинации заполнены одним символом - победа
- Если свободных клеток не осталось и победителя нет - ничья

### Особенности реализации

- Игра полностью на чистом JavaScript без библиотек
- Состояние игры хранится в массиве board
- Визуальное поле и логика синхронизированы

## Игровая механика

### Как определяется победитель

1. После каждого хода вызывается `checkResult()`
2. Перебираются все 8 выигрышных комбинаций из `winConditions`
3. Если три клетки комбинации совпадают и не пустые - победа

### Как обрабатываются клики

1. Пользователь кликает на клетку
2. `handleCellClick()` получает индекс клетки через `data-index`
3. Проверяется: клетка пуста и игра активна
4. Символ записывается в массив `board` и отображается в клетке
5. Вызывается проверка результата

### Как работает перезапуск

1. `currentPlayer` сбрасывается на "X"
2. Массив `board` заполняется пустыми строками
3. `gameActive` устанавливается в `true`
4. Все клетки визуально очищаются

## Структура данных

### Массив board

- Массив из 9 элементов, каждый соответствует клетке поля
- Пустая строка `""` - клетка свободна
- `"X"` или `"O"` - клетка занята соответствующим игроком

### Условия победы

- Хранятся в массиве `winConditions`
- Каждый элемент - массив из 3 индексов клеток
- Всего 8 комбинаций: 3 горизонтали, 3 вертикали, 2 диагонали

### Состояние игры

- `currentPlayer` - чей сейчас ход
- `board` - текущее состояние поля
- `gameActive` - активна ли игра

## Алгоритмы

### Алгоритм проверки победы

1. Перебрать все 8 комбинаций из `winConditions`
2. Для каждой взять три индекса a, b, c
3. Если `board[a]` не пустой и равен `board[b]` и `board[c]` - победа

### Алгоритм определения ничьи

1. Если победителя нет
2. Проверить, есть ли в `board` пустые строки через `board.includes("")`
3. Если пустых нет - ничья

### Алгоритм смены игрока

1. Если нет победителя и нет ничьей
2. `currentPlayer = currentPlayer === "X" ? "O" : "X"`
3. Обновить статус на экране
